== Description ==

Youtube Responsive Iframe Gutenberg block allows users to easily add a youtube video that is responsive and also allows them to choose the aspect ratio of the player to fit their video.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
